## 📖 Getting Started

### For Testing Locally:

1. **Clone Repository**
   ```bash
   git clone https://github.com/PAVAN-18-CODER/aiwithbaaluupdates.git
   cd aiwithbaaluupdates
   ```

2. **Install Backend Dependencies**
   ```bash
   cd backend
   npm install
   ```

3. **Start Backend Server**
   ```bash
   npm start
   # Server runs on http://localhost:3000
   ```

4. **Open Frontend**
   ```bash
   # Open pa1.html in your browser
   file:///path/to/pa1.html
   ```

### For Cloud Deployment:

#### Deploy Backend to Render (5 minutes):
1. Go to https://render.com
2. Sign up with GitHub
3. Create Web Service from repository
4. Set build: `cd backend && npm install`
5. Set start: `cd backend && node server.js`
6. Deploy!

#### Frontend Already Live on GitHub Pages! ✅
