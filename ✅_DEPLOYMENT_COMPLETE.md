## 🐛 Troubleshooting

### Website Not Loading?
1. Wait 3-5 minutes after deployment
2. Check GitHub Actions for errors
3. Hard refresh: Ctrl + Shift + R
4. Try incognito/private mode

### Mobile Menu Not Working?
1. Clear browser cache
2. Check JavaScript is enabled
3. Try different browser
4. Check console for errors (F12)

### News Not Updating?
1. Currently using demo API key (limited)
2. Get free key: https://newsapi.org/register
3. RSS fallback is working automatically
4. See `GET_NEWS_API_KEY.md` for setup
