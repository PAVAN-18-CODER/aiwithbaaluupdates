## 🆘 IF SOMETHING GOES WRONG

### Fastest Fixes

**Website shows blank page?**
→ Check browser console (F12): Look for errors
→ Most likely: Wrong API URL in pa1.html
→ Fix: Update pa1.html, push again

**API endpoint returns 404?**
→ Check Render dashboard: Service shows "Live"?
→ Verify your URL is correct
→ Check Render logs for error

**GitHub Pages not updating?**
→ Check GitHub Actions: Latest push shows green ✓?
→ If red ✗: Click to see build error
→ Fix the error, push again
